HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer element represents additional content such as sitemap, authors and contact details.
  • Tags h1 to h6 define headings
  • Tag p deifnes paragraph
  • Tag br defines line break

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can use *{…} to apply style to all elements
  • We can use tag names (header, footer{…} ) to apply to all these elements
  • We can use classes ( .ABC{…} )to create a class and assign it to particular element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin ... receives up-to-date information
  • git add -A adds all changes to a commit
  • git push commits all the changes to current branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Semicolon ; can be avoided if there is only 1 operation on the line
  • Arrays can store different types of data: integers, strings, boolean etc.
  • JS code can be imported into HTML file using script tag
